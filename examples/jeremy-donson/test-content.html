Mad Libs is an game from the 1970s which allows players to insert random words in sentences driven by clues indicating part of speech.

The results are grammatically accurate sentences with maximum silliness.

    This open source tool allows authors to generate customized html forms.
  • Type into this first box of three.
  • View the results in html and browser output (windows 2 + 3).
  • Copy the html text when you are ready to paste and format it in a testMadLibs.html file.
  • This is a good tool to learn prior to learning about templating and/or forms.
    What I like about this open source Mad Libs game:
  • Open source: https://github.com/seport/Mad-Libs-Generator
  • Avoids formatting text.
  • Supports character sets other than English.
